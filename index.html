Import Tips:

Note: Imported images should be pixel art, and make sure your image is clear (not blurry) and has simple 2D colors only.

You can:

  1. Use the cropping tool to ensure that all pixel grids in the image are complete.
  2. Adjust the sampling grid size to make sure each pixel in the original image corresponds closely to the sampling grid.
  3. After roughly aligning the grid, you can toggle the sampling dots to check if the position of each sampling point is correct.
  4. Click Generate Pixels to check it on canvas.
